# Projeto de Análise Exploratória de Dados

Este projeto foi desenvolvido como atividade acadêmica, com o objetivo de realizar uma análise exploratória de dados utilizando R e RMarkdown.

## 📊 Base de dados
Lifestyle and Health Risk Prediction Dataset (Kaggle)

## 📌 Objetivos
- Analisar variáveis numéricas
- Verificar normalidade dos dados
- Avaliar completude
- Aplicar imputação com MICE
- Criar dashboard interativo com Shiny

## 🛠 Tecnologias utilizadas
- R
- RMarkdown
- Shiny
- ggplot2
- summarytools
- ggpubr
- mice

## 📁 Arquivos
- EDA_PD_Carla Assis.Rmd
- app.R
- Dataset
- Print do dashboard
